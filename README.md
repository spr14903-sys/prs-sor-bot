# PRS SOR Pricing Agent

Telegram Bot + Web App for UK Social Housing SOR code pricing.

## Features
- 🤖 Telegram Bot - send job descriptions, get SOR codes + prices
- 💬 Web Chat - conversational SOR pricing assistant  
- 🔍 SOR Code Search - search 3,580 codes
- 📋 Quote Builder - build and export quotations
- 🇷🇴🇬🇧 Romanian + English support

## Environment Variables
Set these in Railway:
- `TELEGRAM_TOKEN` - from @BotFather
- `ANTHROPIC_API_KEY` - from console.anthropic.com

## Deploy to Railway
1. Push to GitHub
2. Connect Railway to GitHub repo
3. Set environment variables
4. Deploy!
